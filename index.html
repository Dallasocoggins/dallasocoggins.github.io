Calamity Crew

2024

Calamity Crew is a cooperative multiplayer game developed with the LIVE Lab, where players work together to rescue survivors trapped under rubble on a flooding island. As the water rises, teamwork and quick decision-making become essential to navigate the environment and save as many people as possible before time runs out. I played a key role in developing the game's peer-to-peer multiplayer system, ensuring smooth and responsive online gameplay. Through a combination of engaging cooperative mechanics and polished networking, **Calamity Crew** delivers an immersive multiplayer experience that challenges players to work together under pressure.

Reliable Data Transfer over UDP

Apr 2024

Reliable Data Transfer over UDP is a CSCE 463, Networks and Distributed Processing, project in which I implemented a custom Reliable Data Transfer (RDT) protocol on top of UDP to ensure ordered, lossless communication between networked systems. Unlike TCP, UDP does not guarantee reliable delivery, so this project involved designing and implementing retransmission mechanisms using acknowledgments and sequence numbers to handle packet loss, corruption, and out-of-order delivery. To test the protocol's resilience, I simulated network conditions with varying levels of packet loss and latency, refining the retransmission logic for efficiency. This project deepened my understanding of low-level networking, socket programming, and transport-layer protocols, reinforcing key concepts in network reliability and performance optimization.

C++ Web Crawler

Feb 2024

For CSCE 463, I developed a high-performance web crawler capable of processing URLs from an input file and asynchronously fetching web content. The crawler was designed with parallelized HTTP requests using multithreading, significantly improving page retrieval efficiency. The project enhanced my understanding of low-level networking, concurrent programming, and distributed systems, providing hands-on experience with socket programming and HTTP communication protocols.
