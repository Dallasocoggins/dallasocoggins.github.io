Networking & Systems

Reliable Data Transfer over UDP

Apr 2024

Reliable Data Transfer over UDP is a CSCE 463, Networks and Distributed Processing, project in which I implemented a custom Reliable Data Transfer (RDT) protocol on top of UDP to ensure ordered, lossless communication between networked systems. Unlike TCP, UDP does not guarantee reliable delivery, so this project involved designing and implementing retransmission mechanisms using acknowledgments and sequence numbers to handle packet loss, corruption, and out-of-order delivery. To test the protocol's resilience, I simulated network conditions with varying levels of packet loss and latency, refining the retransmission logic for efficiency. This project deepened my understanding of low-level networking, socket programming, and transport-layer protocols, reinforcing key concepts in network reliability and performance optimization.

C++ Sockets Networking Threads

PlayerAnalyticsPlugin (Unreal Engine 5)

Jan 2025 - May 2025

For my Computer Science capstone, with a team of four others, I developed PlayerAnalyticsPlugin, a configurable Unreal Engine 5 plugin that collects player data and transmits it to a remote server for analytics purposes. The plugin reads a local JSON file containing player metrics and sends it to a Flask backend using HTTP requests. This adds the JSON files to a MongodDB databases which can be accessed through a variety of GET request endpoints to be sent to a AWS amplify frontend. Server IP and authentication tokens are easily configurable within the Unreal Project Setting.

Designed with modularity and ease of integration in mind, this tool allows developers to track gameplay behavior, player stats, and other custom data points during play sessions - enabling powerful real-time feedback and iterative design workflows.
